VHNet v2 Architecture

Dual-Branch Multi-Task Inference

An end-to-end U-Net framework that concurrently optimizes high-resolution RGB reconstruction and categorical terrain classification. By sharing a feature encoder, VHNet v2 maintains structural consistency across both spectral and semantic domains.

Inference Pipeline Flow

1. IR Telemetry Ingestio

Raw, single-channel infrared input normalization.

↓

2. Shared Encoder

Deep convolutional feature extraction for structural and textural cues.

↓

3. Task Bifurcation

Latent representation split into parallel branches for specialized processing.

↓

4A. RGB Reconstruction

Spatial super-resolution utilizing PixelShuffle to generate high-fidelity true-color outputs.

+

4B. Semantic Interpretation

Categorical terrain mapping using Cross-Entropy optimization.

↓

5. Multi-Task Convergence

Synchronized optimization of perceptual and structural losses for maximum model stability.

↓

Final Output

Simultaneous generation of a high-resolution True-Color image.

System Components

Input Normalization

Pre-process raw thermal/infrared tiles to ensure uniform tensor scaling before feeding the neural network.

Shared Encoder

Efficiently compute deep feature maps once, saving computational overhead while serving both downstream tasks.

Multi-Task Optimization

Prevent gradient explosions by mathematically balancing perceptual RGB loss with categorical cross-entropy.

Flask API Backend

Serve the compiled PyTorch `.pth` weights through a high-speed Python REST API for real-time web inference.
